## Descripcion del proyecto
Visualización sobre Cansancio Digital y Análisis de Productividad, para encontrar los causantes
de el cansancio digital y analizar la productividad de esta mediante el uso de diferentes variables, identificando cuales tienen un mayor modos de trabajos, trabajos, etc. Tienen o se ven afectados por este cansancio y si la productividad se ve afectada por este. Analizando los resultados observamos como la modalidad de trabajo Remota contiene el mayor numero de Fatiga mental por parte de los usuarios.
